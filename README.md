# InfluencerMatchAI

## Overview

InfluencerMatchAI is a machine learning-based influencer recommendation system that helps businesses find suitable micro-influencers based on budget and category preferences.

## Features

* Synthetic influencer dataset generation
* Data preprocessing and analysis
* K-Means clustering for influencer segmentation
* Recommendation engine
* Flask-based web application
* User-friendly interface

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* HTML
* CSS

## Machine Learning Techniques

* K-Means Clustering
* Nearest Neighbors Recommendation

## Workflow

1. Generate influencer dataset
2. Perform data cleaning and preprocessing
3. Train K-Means clustering model
4. Build recommendation engine
5. Develop Flask web application
6. Display influencer recommendations
